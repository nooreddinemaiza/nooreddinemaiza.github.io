À propos

Deux expertises,
une seule vision

Je suis Nooreddine Maiza, administrateur systèmes & réseaux et développeur web PHP basé au Maroc. Ma démarche est de combiner ces deux domaines : concevoir des solutions web utiles à l'administration réseau et système — interfaces de gestion, outils de supervision, portails d'accès.

Côté infrastructure, j'administre des environnements Windows Client & Server (Active Directory, GPO, WSUS) et Linux (Ubuntu, CentOS, Rocky Linux). Pour les équipements réseau, je maîtrise la configuration des équipements Cisco IOS (routage, VLAN, ACL, NAT) et je progresse activement sur d'autres systèmes.

Côté développement, tous mes projets PHP sont en orienté objet avec système de routage interne — une architecture que j'ai formalisée dans mon framework maison NM Base Framework.
